This is just a simple simulation of very basic inclined plane model. In addition to the slope geometry and friction there is a magnetic drag force possible to be taken under consideration.

The magnetic drag is there to simulate the situation when a permenent magnet is sliding on a conductor slope. The magnetic drag force is assumed to be Fm = dz * velocity

So the general equation for forces is F = m*g(sin(alpha) - mi*cos(alpha)) - Fm

with off course some additional limiatation for the friction drag force presence only if v>0 and so on...
